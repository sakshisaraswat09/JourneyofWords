# Journey of Words ✍️

Journey of Words is a full-stack blogging platform that enables users to create, publish, and manage blog content through an intuitive and responsive interface.

The platform supports user authentication, role-based authorization, content categorization, commenting, and blog management features, providing a complete blogging experience for writers and readers.

## Features

### User Management
- User Registration & Login
- JWT Authentication
- Role-Based Access Control
- Secure Password Storage

### Blog Management
- Create Blog Posts
- Edit Existing Posts
- Delete Posts
- Rich Text Content Editor
- Tag & Category Support

### Reader Features
- Browse Blogs
- Search Posts
- Filter by Category
- Read Full Articles

### Comment System
- Add Comments
- Edit Comments
- Delete Comments
- Admin Moderation

### User Experience
- Responsive Design
- Pagination
- Search Functionality
- Mobile-Friendly Interface

## Installation

### Clone Repository

git clone https://github.com/sakshisaraswat09/JourneyofWords.git

cd JourneyofWords

### Install Backend Dependencies

npm install

### Configure Environment Variables

MONGODB_URI=
JWT_SECRET=

### Start Server

npm start
